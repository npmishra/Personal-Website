# Modern Multi-Page Portfolio Website

A fully responsive, modern portfolio website for Narayan Prasad Mishra - Creative Graphics Designer & Web Developer.

## Features

- **6 Complete Pages**: Home, About, Services, Gallery, Blog, and Contact
- **Modern UI Design**: Clean, colorful, and professional interface
- **Fully Responsive**: Works on all devices (desktop, tablet, mobile)
- **Local Image Support**: All images loaded from local assets folder
- **Interactive Elements**: Animations, form validation, gallery filtering
- **Optimized Performance**: Fast loading and smooth transitions
- **SEO Friendly**: Proper meta tags and semantic HTML
